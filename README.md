# killjoys

## make some noise

`killjoys` is an early-stage Go project for a terminal-first music player with an MCR-inspired identity.

The goal is to build something raw, sharp, and personal instead of a generic player: a small app that treats the terminal like a stage, not just a shell.

Right now the project is still taking shape, but the direction is clear: Go, a strong visual mood, and a player that feels like `Killjoy` from the very beginning.
